    • CSS

    CSS Csignifica hojas de estilo en cascada. Es un lenguaje de hojas de estilo que se utiliza para describir la apariencia y el formato de un documento escrito en lenguaje de marcado. Proporciona una función adicional a HTML. Generalmente se usa con HTML para cambiar el estilo de las páginas web y las interfaces de usuario. También se puede utilizar con cualquier tipo de documento XML, incluidos XML sin formato, SVG y XUL.

    CSS se usa junto con HTML y JavaScript en la mayoría de los sitios web para crear interfaces de usuario para aplicaciones web e interfaces de usuario para muchas aplicaciones móviles.
