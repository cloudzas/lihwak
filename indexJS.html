    JavaScript es un lenguaje de secuencias de comandos basado en objetos que es ligero y multiplataforma.
    JavaScript no es un lenguaje compilado, pero es un lenguaje traducido. El traductor de JavaScript (incrustado en el navegador) es responsable de traducir el código JavaScript para el navegador web.



    ¿Qué es JavaScript?

    JavaScript (js) es un lenguaje de programación ligero orientado a objetos que utilizan varios sitios web para crear secuencias de comandos en las páginas web. Es un lenguaje de programación completo e interpretado que permite la interactividad dinámica en sitios web cuando se aplica a un documento HTML. Fue introducido en el año 1995 para agregar programas a las páginas web en el navegador Netscape Navigator. Desde entonces, ha sido adoptado por todos los demás navegadores web gráficos. Con JavaScript, los usuarios pueden crear aplicaciones web modernas para interactuar directamente sin tener que recargar la página cada vez. El sitio web tradicional usa js para proporcionar varias formas de interactividad y simplicidad.
    Aunque, JavaScript no tiene conectividad con el lenguaje de programación Java. El nombre fue sugerido y proporcionado en los tiempos en que Java estaba ganando popularidad en el mercado. Además de los navegadores web, las bases de datos como CouchDB y MongoDB utilizan JavaScript como lenguaje de secuencias de comandos y consultas.




    Historia de JavaScript

    En 1993, apareció Mosaic, el primer navegador web popular. En el año 1994, Netscape fue fundado por Marc Andreessen. Se dio cuenta de que la web necesitaba volverse más dinámica. Por lo tanto, se creía que se proporcionaba un "lenguaje adhesivo" a HTML para facilitar el diseño web a los diseñadores y programadores a tiempo parcial. En consecuencia, en 1995, la empresa contrató a Brendan Eich con la intención de implementar e integrar el lenguaje de programación Scheme en el navegador. Pero, antes de que Brendan pudiera comenzar, la empresa se fusionó con Sun Microsystems para agregar Java a su Navegador para poder competir con Microsoft en las tecnologías y plataformas web. Ahora, dos lenguajes estaban allí: Java y el lenguaje de programación. Además, Netscape decidió dar un nombre similar al lenguaje de secuencias de comandos como el de Java. Condujo a 'Javascript'. Finalmente, en mayo de 1995, Marc Andreessen acuñó el primer código de Javascript llamado 'Mocha'. Posteriormente, el equipo de marketing reemplazó el nombre por 'LiveScript'. Pero, debido a razones de marca comercial y algunas otras razones, en diciembre de 1995, el idioma finalmente se renombró como 'JavaScript'. A partir de entonces, nació JavaScript.

    Caracteristicas de JavaScript

    Estas son las siguientes características de JavaScript:

    1. Todos los navegadores web populares admiten JavaScript, ya que proporcionan entornos de ejecución integrados.
    2. JavaScript sigue la sintaxis y la estructura del lenguaje de programación C. Por lo tanto, es un lenguaje de programación estructurado.
    3. JavaScript es un lenguaje débilmente tipificado, donde ciertos tipos se convierten implícitamente (dependiendo de la operación).
    4. JavaScript es un lenguaje de programación orientado a objetos que usa prototipos en lugar de usar clases para la herencia.
    5. Es un lenguaje ligero e interpretado.
    6. Es un lenguaje que distingue entre mayúsculas y minúsculas.
    7. JavaScript es compatible con varios sistemas operativos, incluidos Windows, macOS, etc.
    8. Proporciona un buen control a los usuarios sobre los navegadores web.

    Aplicación de JavaScript

    JavaScript se utiliza para crear sitios web interactivos. Se utiliza principalmente para:

    • Validación del lado del cliente,
    • Menús desplegables dinámicos,
    • Mostrando fecha y hora,
    • Mostrar ventanas emergentes y cuadros de diálogo (como un cuadro de diálogo de alerta, un cuadro de diálogo de confirmación y un cuadro de diálogo de solicitud),
    • Mostrar relojes, etc.
