    La creación de un sitio web se puede hacer de diferentes maneras. Puede optar por utilizar creadores de sitios web como Wix o WordPress, o codificar su propio sitio y luego encontrar un host para él. La programación de su propio sitio web requiere ciertas habilidades informáticas, incluido el desarrollo informático y los idiomas.

    Nuestra primera versión estará dedicada a diferentes lenguajes: HTML,CSS y una introducción a JavaScript. presentamos los principales principios y funcionalidades (cuidadosamente traída por nosotros y referenciadas). Este contenido pretende aportar nuestros primeros conocimientos adquiridos así como herramientas. esto te permitirá no solo crear pequeños proyectos como lo hacemos nosotros, sino también seguir nuestro progreso y hacerte más eficiente si necesitas profundizar más tarde en tal o cual lenguaje.



    HTML para la base de su sitio web

    El primero de los lenguajes que debes saber para codificar tu sitio web es HTML, que significa Hypertext Markup Language. Este lenguaje es la base de todas las páginas web. Este último se utiliza para indicar qué texto mostrar. El código HTML se utiliza para crear la estructura del sitio, pero también el contenido del sitio.



    CSS para el diseño de sitios web

    Hacer una página web simple para mostrar texto e imágenes a menudo no es suficiente para hacer un sitio acogedor y moderno. Una vez que se ingresa el texto y se colocan las imágenes, también es importante diseñar el sitio web eligiendo las fuentes, el color del fondo de pantalla, la posición de los elementos. Es particularmente con el lenguaje CSS que puedes pasar de un conjunto de información sin estilo a un sitio web un poco más elaborado y jerárquico.



    JS para crear interacción

    Javascript se utiliza en el desarrollo web para crear páginas con interacciones de usuarios. Ya no se trata de procesar datos sino de animar la página con, por ejemplo, un mensaje de agradecimiento que aparece cuando el visitante se suscribe a la newsletter, o una ventana emergente que llega cuando el visitante mueve el cursor y sale de la página. campo del sitio web, etc.
